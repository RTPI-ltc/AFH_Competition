# 京东超级品牌日（JD Super Brand Day）规则知识库

> **聚焦赛题方向 C**：仅整理"可被解析为检查项 + 风险点"的规则条款。京东与天猫的核心差异在于：**价格回溯口径 14 天而非 30 天**、**价保 30 / 45 天**、**自营 vs POP 双轨**。

---

## 一、商家侧：可被解析为检查项的规则条款

### 1.1 资质门槛

| 检查项 | 门槛 |
|--------|------|
| 经营主体 | 京东官方旗舰店、官方专卖店；C 类店铺不可参加 |
| 行业资质 | 贵金属生产/销售许可、商标证 |
| 店铺等级 | 京东商智综合分 ≥ 9.0 |
| 违规记录 | 近 90 天无 A 类违规 |
| 经营时长 | 旗舰店 ≥ 6 个月 |
| 物流入仓 | 推荐入驻 JD Logistics 仓（影响 211 时效标签） |

### 1.2 自营 vs POP 双轨

| 维度 | 自营（FBP / 经销） | POP 旗舰店（SOP） |
|------|-------------------|--------------------|
| 商品所有权 | 京东 | 商家 |
| 价格制定权 | 京东买手 + 品牌方提报"京采价"（含税供货底价 + 毛保返点） | 商家自定，但需符合"京采价"对标 |
| 物流 | 京东仓配 211 | 商家发货 / 入仓 JDL |
| 售后 | 京东客服 | 商家客服 |

> Agent 应在解析规则时先确认 SKU 走的是自营还是 POP；不同通道价格管控、库存逻辑差异显著。

### 1.3 SKU 选品条件

| 检查项 | 门槛 |
|--------|------|
| 报名 SKU 数 | ≥ 150 款 |
| 超级单品数 | ≥ 3 款 |
| 京东独家首发 / 联名 SKU | ≥ 1 款 |
| 单 SKU 近 90 天好评率 | ≥ 97%（京东比天猫严） |
| 商品证书电子件 | CMA 或国检任一可在线核验 |
| 参数精度（黄金/镶嵌） | 与天猫一致（克重 0.01g、4C 精度到 0.01ct + 完整 4C 等级） |

### 1.4 价格管控（核心）

| 检查项 | 规则 |
|--------|------|
| 京采价 vs 全网 | 必须 ≤ 全网最低价（含天猫、抖音、品牌官网） |
| 活动价 vs 店内 | ≤ **近 14 天**京东自营 / POP 最低成交价（**注意：京东 14 天，不是 30 天**） |
| 大促额外让利 | 部分品类要求活动价 ≤ 历史 365 天最低 × 95% |
| 划线价合规 | 必须近 **14 天**内有实际成交、未划线的销售价 |
| 价保期 | **30 天**（普通用户）/ **45 天**（PLUS 用户） |
| 百亿补贴价 | 进入百亿补贴池的 SKU 须 ≤ 近 90 天历史最低 |

### 1.5 品类互斥规则

| 互斥项 | 说明 |
|--------|------|
| 跨店满减 | 默认参加"每满 200 减 30"（大促升级"每满 1000 减 100"） |
| 百亿补贴 SKU | 默认不再参加跨店满减 |
| 国补 / 政府补贴 SKU | 不可叠加平台券（可叠加店铺券） |
| 同期活动 | 同 SKU 不可同时报"超级品牌日"和"超级品类日" |
| PLUS 价 | 可与跨店满减叠加，但不可与品牌定向券叠加 |

---

## 二、消费者侧（仅商家需要执行的合规条款）

| 商家必须执行 | 说明 |
|-------------|------|
| 价保 30 / 45 天 | 普通用户 30 天，PLUS 用户 45 天，按到手价口径退差 |
| PLUS 价不要求"全网最低"但要"会员中最低" | PLUS 价被外渠抓取后用于挑战全网最低承诺，是高频风险 |
| 国补 SKU 不可叠加平台券 | 商详页必须明示"国补 SKU"不参与跨店满减 |

---

## 三、商家关注的流程与环节（赛题视角）

| 阶段 | 关键检查项 |
|------|-----------|
| T-30 京采价盘整 | 含税成本 + 行业返点 + 京东毛保 → 京采底价 |
| T-30 价格盘点 | 拉 SKU 近 **14 天** + 近 90 天京东历史成交价（注意区别于天猫 30 天） |
| T-15 京东商家后台提报 | 上传 SKU、京采价、库存承诺、证书 |
| T-7 PLUS 触达 | 京东 APP Push、京麦短信、PLUS 微信社群 |
| T0 物流时效 | 入仓 JDL，确保 211 时效；超 24 小时未发即扣分 |
| T+30/T+45 价保退差 | T+1 起每日跑批；PLUS 用户特别关注 45 天窗口 |

---

## 四、常见踩坑（与天猫规则差异 + 京东独有项）

| 风险点 | 触发场景 | 风险等级 |
|--------|---------|---------|
| 14 天 vs 30 天混淆 | 用天猫 30 天口径填京东 | high |
| 京采价被穿底 | 供货价高于外渠成交价 | high |
| 百亿补贴叠加跨店满减 | 误以为可叠加 | high |
| 国补 SKU 叠加平台券 | 政府返款被取消 | high |
| PLUS 价低于全网 | 被抓取后违反全网最低承诺 | medium |
| 物流时效不达 | 未入仓 JDL，超 24h 未发 | high |
| 自营 / POP 同款双轨穿仓 | 自营价 < POP 价或反之 | high |
