# 淘宝聚划算 / 百亿补贴规则知识库

> **聚焦赛题方向 C**：聚划算 / 百亿补贴的核心特征是"以**全网最低价**为前提的流量频道"，价格规则比品牌日更严，回溯窗口为 90 天。

---

## 一、商家侧：可被解析为检查项的规则条款

### 1.1 资质门槛

| 检查项 | 门槛 |
|--------|------|
| 经营主体 | 天猫旗舰店、专卖店、专营店；C 店、企业店不可参加 |
| 店铺评分 | DSR ≥ 4.7 |
| 违规记录 | 近 30 天店铺无 A 级违规 |
| 经营时长 | 旗舰店 ≥ 6 个月 |
| 发货时效 | 入驻菜鸟仓或自建仓，48 小时发货率 ≥ 99% |

### 1.2 SKU 选品条件

| 检查项 | 门槛 |
|--------|------|
| 聚划算品牌闪购单次报名 SKU 数 | ≥ 20 |
| 百亿补贴 | 可单 SKU 报名 |
| 单 SKU 近 90 天店内销量 | ≥ 30 件 |
| 单 SKU 近 30 天好评率 | ≥ 95% |
| 让利幅度（让利 + 券 + 平台补贴合计） | ≥ 15%（不达即不进算法池） |
| 风险品类 | 定制款、刻字款、按需生产款不允许进百亿补贴池 |

### 1.3 价格管控（最严）

| 检查项 | 规则 |
|--------|------|
| 全网最低价 | 必须 ≤ 京东、抖音、得物、唯品会、品牌官网、线下经销商任一渠道价 |
| 店内最低价 | 必须 ≤ **近 90 天**店内最低成交价（**注意：百补 90 天**） |
| 平台直接补贴 | 平台对部分热销 SKU 提供 5–15% 立减券（由阿里出资） |
| 划线价合规 | 必须近 30 天内有实际成交、未划线的销售价 |
| 价保期 | 15 天 |
| 限购 | 单人限购 1–3 件（珠宝品类一般 2 件） |

### 1.4 品类互斥与叠加

| 规则 | 说明 |
|------|------|
| 跨店满减 | **百亿补贴 SKU 默认不参加跨店满减**（已是平台让利后价格） |
| 店铺券 | 可叠加，但券额计入"让利幅度"考核 |
| 88VIP | 可叠加 95 折 |
| 平台超级红包 | 可使用 |
| 同期品牌日 / 超品日 | 不可同期 |
| 聚划算品牌闪购 vs 百亿补贴 | 二选一，不可同 SKU 双报 |

> ⚠️ 阿里风控算法每日抓取京东、抖音、拼多多等平台同款 SKU 价格。**穿底一次 → SKU 立即踢出补贴池且 30 天内不可再申请。**

## 二、消费者侧（仅商家需要执行的合规条款）

| 商家必须执行 | 说明 |
|-------------|------|
| 价保 15 天 | 按到手价口径退差 |
| 限购话术 | 客服必须告知"单人限购 X 件" |
| 跨店满减不叠加 | 商详页明示"百亿补贴 SKU 不参与跨店满减" |

---

## 三、商家关注的流程与环节（赛题视角）

| 阶段 | 关键检查项 |
|------|-----------|
| T-30 全网价格盘查 | 抓取京东、抖音、拼多多、得物、品牌官网同款价；确保"我家最低" |
| T-30 近 90 天店内最低价 | 从生意参谋导出滚动 90 天最低价（注意 90 天） |
| T-30 让利幅度计算 | (划线价 - 活动价) / 划线价 + 券补 + 平台补贴 ≥ 15% |
| T-30 互斥盘点 | 同 SKU 是否在天猫超品日、品类日、行业大促中已报名 |
| T-15 后台提报 | 阿里妈妈"营销中心 → 百亿补贴 → 入驻提报" |
| T+0 每日比价 | 跑批比价，发现穿底立即跟进；穿底 1 次即踢池 |
| T+15 价保 | 每日跑批退差 |

## 四、常见踩坑

| 风险点 | 触发场景 | 风险等级 |
|--------|---------|---------|
| 京东侧穿底 | 京东百亿补贴页同款更低 | high |
| 抖音直播间穿底 | 同款达人直播间限时更低 | high |
| 让利不足 15% | 划线价虚高导致表面让利大、实际不足 | medium |
| 跨店满减误叠加 | 商详页错挂"跨店满减"标签 | high |
| 退货率过高 | > 8% 触发流量降权 | medium |
| 限购规则错配 | 限购 5 件被黄牛刷 | medium |
| 30 天 vs 90 天混淆 | 用天猫 30 天填百补 90 天 | high |
