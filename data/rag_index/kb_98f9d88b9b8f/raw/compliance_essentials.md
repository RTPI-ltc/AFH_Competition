# 价格与商品标注合规要点（Compliance Essentials）

> **聚焦赛题方向 C**：仅保留"与电商活动规则执行直接相关"的合规依据——价格管控合规（《明码标价规定》）+ 商品标注国标（GB 11887、GB/T 16554、GB/T 16552）。
> 已剥离《广告法》《反不正当竞争法》《消费者权益保护法》《电子商务法》的法规罗列、假一赔十等售后赔付制度、RJC 等供应链合规体系——这些属于赛题方向 D（风控合规）范畴。

---

## 1. 价格合规（活动价 / 划线价的国家口径）

### 1.1 《明码标价和禁止价格欺诈规定》（国家市场监督管理总局令第 56 号，2022 年 7 月起施行）

**与电商活动规则直接相关的核心条款**：

| 条款要点 | Agent 解析含义 |
|---------|----------------|
| 划线价必须是"促销前 7 日内最低成交价" | "近 7 天" 是市监总局口径；天猫加严到 30 天，京东加严到 14 天 |
| 若 7 日内无成交记录，按"最后一次成交价"标注 | 新 SKU 没有 7 天内成交 → 不可标划线价 |
| 不得使用"建议零售价 / 厂商指导价 / 吊牌价"作为划线价 | 商详页严禁用 MSRP 当划线价 |
| 不得虚高原价误导消费者 | 最高 50 万元罚款 + 行政警告 |

### 1.2 各平台对划线价的额外加严

| 平台 | 划线价口径 |
|------|-----------|
| 市监总局基线 | 近 **7 天**有实际成交 |
| 天猫 | 近 **30 天**有实际成交 |
| 京东 | 近 **14 天**有实际成交 |
| 抖音商城 | 近 **30 天** + 行业大促期间再加 7 天保护期 |

> Agent 用法：解析"划线价"相关规则时**按最严平台口径**校验；规则原文若仅说"原价"，Agent 应主动澄清是按哪个平台口径。

### 1.3 常见违规模式

| 模式 | 示例 | Agent 风险等级 |
|------|------|----------------|
| 用 MSRP 当划线价 | 划线价 9999（厂商指导价），实际从未以该价销售 | high |
| 虚高吊牌价 | 5 年前吊牌价 9999，近 30 天最高成交 399 | high |
| 数据时窗错配 | 用近 365 天最高价当划线价（按 7/14/30 天口径都违规） | high |
| 满赠估值超 30% 触发回溯 | 满赠中赠品价值 > 商品 30% → 触发近 30 天最低价重算 | medium |

## 2. 黄金 / 铂金 / 银饰命名合规

### 2.1 GB 11887-2012《首饰 贵金属纯度的规定及命名方法》

**强制性国标，Agent 解析涉及"成色 / 纯度"的规则时按此校验。**

| 标识 | 含金量 | 通俗名 | 必备印记 |
|------|--------|--------|---------|
| Au990 | ≥ 990‰ | 足金 | 足金 / G990 / Au990 |
| Au999 | ≥ 999‰ | 足金 999 | 足金 999 / Au999 |
| Au999.9 | ≥ 999.9‰ | （旧称千足金，2016 起删除） | Au999.9 |
| Au999.99 | ≥ 999.99‰ | （俗称万足金，2016 起删除） | Au999.99 |
| Au750 | 75.0% ± 0.3% | 18K 金 | Au750 / 18K |
| Au585 | 58.5% ± 0.3% | 14K 金 | Au585 / 14K |
| Pt950 | ≥ 950‰ | 铂金 / 950 铂金 | Pt950 / Plat950 |
| Pt990 | ≥ 990‰ | 990 铂金 | Pt990 |
| S925 | ≥ 925‰ | 925 银 / Sterling Silver | S925 / Ag925 |

### 2.2 GB 11887-2012 第 1 号修改单（2016 年起施行）

- **删除"千足金 / 万足金 / 千足银"等称谓**，统一使用 Au999、Au999.9 等具体成色。
- ✅ 合规商详页：足金 999.9 或 Au999.9。
- ❌ 不合规商详页：仅写"千足金"作为成色标识。
- Agent 在解析"千足金"相关规则时**主动澄清**："本规则中千足金是指 Au999 还是 Au999.9？"

### 2.3 印记 / 钢印（电商商详页必查）

国标要求所有贵金属饰品必须打钢印，缺失即"三无产品"：

| 必须包含 | 示例 |
|---------|------|
| 厂家代号 | CTF / 周大福 |
| 材料 + 纯度 | 足金 / Au999 / 18K / Pt950 |
| 镶嵌主石（如有） | DIA / 0.50ct |

## 3. 钻石 / 玉石 / 珍珠 命名合规

### 3.1 GB/T 16552-2017《珠宝玉石 名称》（推荐性国标）

- 翡翠：必须明示 A 货 / B 货 / C 货 / B+C 货 / 优化处理；B/C 货当 A 货卖构成欺诈。
- 钻石：必须明示天然钻 / 培育钻（实验室钻 / Lab-grown）；混淆构成欺诈。
- 珍珠：必须明示海水 / 淡水。
- 玉石：和田玉 / 软玉的产地（新疆 / 青海 / 俄罗斯 / 韩国）需有独立产地溯源证书才能标注。

### 3.2 GB/T 16554-2017《钻石分级》（推荐性国标）

- 4C 分级方法与 GIA 体系一致。
- 30 分以上钻石商详页必须填写完整 4C（Carat / Color / Clarity / Cut）+ 证书编号。
- 异形钻不评 Cut 等级，商详页禁标"3EX"。

### 3.3 GB/T 18781-2008《珍珠分级》（推荐性国标）

- 七项指标：直径、形状、颜色、光泽、表皮、珠层、匹配。
- 光泽分级"极强 / 强 / 中 / 弱"不可虚标。

## 4. 权威机构（证书核验的合法来源）

| 机构 | 简称 | 权威 | 平台接受度 |
|------|------|------|-----------|
| 国家珠宝玉石质量监督检验中心 | NGTC / 国检 | 国内最高 | 所有平台必查 |
| Gemological Institute of America | GIA | 国际钻石权威 | 30 分以上钻石必备 |
| International Gemological Institute | IGI | 国际，培育钻常用 | 部分平台接受 |
| Hoge Raad voor Diamant | HRD | 欧洲钻石权威 | 部分平台接受 |
| 中国计量认证 | CMA | 实验室检验资质 | 电商抽检报告必备 |
| 中国合格评定国家认可委员会 | CNAS | 实验室国际互认 | 电商抽检报告补充 |
| 上海黄金交易所 | SGE | 黄金价格基准（Au9999） | 计重金"基础金价"来源 |

> Agent 用法：解析"GIA 钻戒享 X 折"等规则时，从 SKU.certificates 中筛选 `type='GIA'`；不存在 → 输出不通过。

## 5. Agent 应用示例

### 5.1 价格合规判定

```text
[规则] "活动价不得高于近 30 天最低价"（天猫超品日）
[Agent 解析]
- 规则口径：天猫 30 天滚动窗口
- 引用合规依据：《明码标价和禁止价格欺诈规定》+ 天猫近 30 天加严
- 校验：从订单库拉取该 SKU 近 30 天最低成交价
- 若活动价 > 该值 → high；建议调价或退出报名
```

### 5.2 商品标注合规判定

```text
[规则] "活动期间销售 30 分以上钻戒须公示 4C"（京东超品日）
[Agent 解析]
- 引用合规依据：GB/T 16554-2017
- 校验：30 分以上 SKU 的 certificates 中 GIA 报告是否包含完整 4C 字段
- 缺任一字段 → medium；建议补齐后再上架
```

### 5.3 成色命名歧义澄清

```text
[规则] "千足金商品 9.5 折"
[Agent 解析]
- "千足金"在 2016 国标修改单后已删除
- clarification_question: "本规则中'千足金'指 Au999 还是 Au999.9？"
- confidence: 0.65 → human_review_gate 等待人工答复
- 路由：品类经理（见 ../chow_tai_fook/human_review_routing.md）
```
