# 珠宝行业术语字典（Industry Terminology）

> Agent 在 `rule_parser` 中遇到行业术语时的统一翻译参考。优先匹配本字典中的标准定义；若规则原文与本字典冲突，以原文为准。

---

## 1. 计价 / 定价相关

| 术语 | 定义 |
|------|------|
| **计重款 / 按克卖** | 按克重 × 金价 + 工费定价 |
| **一口价 / 按件卖** | 固定价格，与金价波动无关 |
| **工费** | 加工 / 设计 / 镶嵌的人工与设备成本 |
| **基础金价 / 报价金** | 上海黄金交易所 Au9999 当日价 |
| **品牌金价** | 品牌在基础金价上加价的对外价 |
| **回收金价** | 品牌回收消费者旧金的价格（一般低于品牌金价） |
| **以旧换新** | 旧金抵扣新款价款（按克 + 手续费） |
| **退翻新** | 旧金返工厂翻新，不换款 |
| **足金 / 千足金 / 万足金** | 见 gold_basics.md |

## 2. 工艺与品类

| 术语 | 定义 |
|------|------|
| **古法金** | 借鉴古代工艺（錾刻 / 花丝）的足金，工艺名 |
| **5G 黄金** | 第五代电铸工艺足金 |
| **硬足金** | 微量合金添加，硬度提升的足金 |
| **彩金** | 含色合金的 K 金（玫瑰金、白 K 金、黑金等） |
| **素金** | 不含主石的纯金属饰品 |
| **镶嵌款** | 含主石（钻石、宝石、玉石）的饰品 |
| **求婚钻戒** | 30 分以上单颗钻石镶嵌的戒指 |
| **婚戒 / 对戒** | 婚礼用素金或微镶对戒 |
| **婴儿首饰** | 5g 以下的小件 |
| **金宝** | 投资类金条 |

## 3. 鉴定与认证

| 术语 | 定义 |
|------|------|
| **国检** | 国家珠宝玉石质量监督检验中心（NGTC） |
| **CMA** | 中国计量认证（实验室资质） |
| **CNAS** | 中国合格评定国家认可委员会 |
| **GIA** | 美国宝石学院（钻石鉴定权威） |
| **IGI** | 国际宝石学院（培育钻常用） |
| **HRD** | 比利时钻石高层议会 |
| **A 货 / B 货 / C 货** | 翡翠等级（A 货是天然未处理） |
| **3EX 切工** | 钻石的 Cut / Polish / Symmetry 三项均为 Excellent |
| **荧光（Fluorescence）** | 钻石在紫外光下发光，分 None / Faint / Medium / Strong / Very Strong |

## 4. 平台与电商术语

| 术语 | 定义 |
|------|------|
| **超品日 / SBD** | 超级品牌日（天猫 / 京东主推的品牌专场） |
| **品类日** | 行业类目专场（如珠宝品类日） |
| **聚划算** | 阿里旗下低价心智频道 |
| **百亿补贴** | 阿里 / 京东 / 拼多多的全网最低价频道 |
| **京采价** | 京东自营内部供货价 |
| **划线价** | 商详页带删除线的"原价" |
| **活动价** | 活动期间的成交价 |
| **到手价** | 用户实付价（含跨店满减、券、红包后） |
| **跨店满减** | 跨店铺合并计算的满减优惠 |
| **价保** | 价保期内若降价，平台 / 商家自动退差 |
| **退差** | 价保自动退还差额 |
| **DSR** | Detailed Seller Rating，店铺评分（描述 / 服务 / 物流） |
| **PLUS / 88VIP** | 平台付费会员 |
| **京准通 / 阿里妈妈** | 平台站内广告投放系统 |
| **UD / 全域** | 阿里妈妈站外投放 |
| **达摩盘 / 数据银行** | 阿里人群标签系统 |

## 5. 库存与供应链

| 术语 | 定义 |
|------|------|
| **ERP** | 企业资源计划（核心系统） |
| **OMS** | 订单管理系统 |
| **WMS** | 仓库管理系统 |
| **TMS** | 运输管理系统 |
| **PCS** | Pieces，件数 |
| **SKU** | Stock Keeping Unit，库存单位 |
| **SPU** | Standard Product Unit，标准商品单位 |
| **库存承诺** | 报名活动时承诺的库存数量 |
| **补货周期 / Lead Time** | 从下单到入仓的时间 |
| **现货 / 期货** | 现货为已有库存，期货为按订单生产 |
| **以销定产** | 根据销量预测安排生产 |

## 6. 售后与服务

| 术语 | 定义 |
|------|------|
| **7 天无理由退货** | 收到货 7 天内无理由退换 |
| **30 天质量问题包退包换** | 质量问题 30 天内包退包换 |
| **假一赔十** | 珠宝品类的假货赔付承诺 |
| **以旧换新** | 旧饰品抵扣新饰品价款 |
| **退翻新** | 旧饰品翻新，不换款 |
| **保养** | 镀层翻新、抛光、清洁等服务 |

## 7. 高频歧义术语（Agent 主动澄清）

| 术语 | 可能解读 1 | 可能解读 2 |
|------|------------|------------|
| **白金** | 铂金（Pt） | 白 K 金（Au750 白） |
| **千足金** | 旧国标 Au999 | 现已不允许单独使用 |
| **古法金** | 工艺类型，仍是 Au999 | 易被误解为新成色 |
| **24K 金** | 理论纯金（实际不存在 100%） | 营销话术 |
| **GIA 钻** | 仅指 GIA 证书的钻石 | 易被误解为高品质钻 |
| **A 货翡翠** | 天然未处理 | 易被误解为最高品质 |
| **南洋珠 / 大溪地** | 海水珍珠 + 产地修饰 | 需有产地溯源 |
