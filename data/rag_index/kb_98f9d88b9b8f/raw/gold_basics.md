# 黄金基础知识（Gold Basics）

> 适用于 Agent 解析黄金类活动规则。涵盖成色、工艺、定价模式、印记、行业惯例。

---

## 1. 黄金成色（国家标准 GB 11887-2012《首饰 贵金属纯度的规定及命名方法》）

### 1.1 标准成色

| 标识 | 含金量（千分数） | 通俗名 | 印记 |
|------|------------------|--------|------|
| Au990 | ≥ 990‰ | 足金 | 足金 / G990 / Au990 |
| Au999 | ≥ 999‰ | 足金 999 / 千足金（旧国标） | 足金 999 / Au999 / G999 |
| Au999.9 | ≥ 999.9‰ | 千足金 / 万足金（俗称） | Au999.9 / G999.9 |
| Au999.99 | ≥ 999.99‰ | 万足金 | Au999.99 / G999.99 |
| Au750 | 含金量 75.0% ± 0.3% | 18K 金 | Au750 / G750 / 18K |
| Au585 | 含金量 58.5% ± 0.3% | 14K 金 | Au585 / G585 / 14K |
| Au417 | 含金量 41.7% ± 0.3% | 10K 金 | Au417 / G417 / 10K |
| Au375 | 含金量 37.5% ± 0.3% | 9K 金 | Au375 / G375 / 9K |

### 1.2 重要修订（2016 年国标更新）

GB 11887-2012 第 1 号修改单（2016 年起施行）将"千足金、万足金"这些表述从国标中**删除**，统一改为"足金"。但商家在商详页仍可标注 Au999、Au999.9、Au999.99 等具体成色。这是 Agent 在解析规则时常见的混淆点。

> ✅ 合规做法：商详页可写"足金 999.9"或"Au999.9"。
> ❌ 不合规做法：仅写"千足金"作为成色标识。

## 2. 黄金的两种定价模式（电商核心）

### 2.1 计重金（按克卖）

- **定价公式**：成交价 = 克重 × 当日金价 + 工费
- **当日金价**：实时从上海黄金交易所（Au9999）拉取，或品牌自定义"品牌金价"（一般在交易所价 + 5–20 元/克）。
- **工费**：分"按克计算"（如 30 元/克）和"按件计算"（如 280 元/件）两种。
- **特点**：价格随金价实时波动，**不适用"近 X 天最低价" 价格管控**（因为价格本就不固定）。
- **电商应用**：商详页通常展示"克重 + 工费 + 实时金价" 三段拆分。
- **常见品类**：素金链、素金手镯、足金小件、投资金条。

### 2.2 一口价金（按件卖）

- **定价**：固定价格，与金价波动无关。
- **特点**：进入"近 X 天最低价" 价格管控；价格管控规则与镶嵌类一致。
- **常见品类**：古法金、5G 黄金、硬足金、节日款（招财猫、福字）、传承系列。
- **争议点**：一口价金折合"克单价" 通常比计重金高 30–80%（含设计 / 工艺溢价），消费者投诉时常用"克单价过高" 作为依据。

### 2.3 计重 vs 一口价对照表

| 维度 | 计重金 | 一口价金 |
|------|--------|----------|
| 定价 | 克重 × 金价 + 工费 | 固定 |
| 价格管控 | 通常不进入 | 进入"近 30 天最低价" |
| 工费占比 | 5–15% | 设计 / 工艺溢价 30–80% |
| 适合活动 | "金价立减" / "工费立减" | 超品日 / 百亿补贴 |
| 退货 | 一般支持 7 天无理由 | 一般不支持 7 天无理由 |
| 以旧换新 | 计重换新（按克结算 + 手续费） | 按工艺类型差异收费 |

## 3. 黄金的工艺类型

### 3.1 古法金

- **定义**：借鉴中国古代金器制作工艺（如錾刻、花丝、镶嵌）的足金饰品，表面通常呈"哑光"或"磨砂"质感。
- **品类标识**：传承系列、宫廷系列。
- **国标**：仍属 Au999 / Au999.9 成色范畴；"古法金" 是工艺名称，不是成色标识。
- **特点**：硬度比普通足金稍高（但仍属软金），工艺溢价大。
- **电商定价**：一般为一口价。

### 3.2 5G 黄金

- **定义**："5G" 是"5th Generation" 缩写，指第五代黄金电铸工艺，比传统硬金硬度更高、更轻、可做细薄结构。
- **国标**：成色仍为 Au999（足金 999），但因电铸工艺，单件克重通常较轻。
- **特点**：可以做出非常细的发丝纹饰、镂空、3D 结构。
- **电商定价**：一口价为主。

### 3.3 硬足金

- **定义**：在足金中添加微量稀土元素（铜、银、钯等不超过 0.1%），提升硬度而不改变成色。
- **国标**：仍属 Au999 成色。
- **特点**：硬度比普通足金高 4 倍以上；适合做尖角设计；耐刮擦。
- **电商定价**：一口价为主。

### 3.4 普通素金 / 计重款

- **定义**：传统熔铸 / 锻造工艺，无明显附加工艺溢价。
- **品类标识**：素金链、素金手镯、素金戒指。
- **电商定价**：计重为主。

### 3.5 工艺类型在商详页的标注要求

- 必须在商详页明示工艺类型（古法金 / 5G 黄金 / 硬足金 / 普通）。
- 不能用"古法金" 替代成色；成色必须独立标注（如 Au999）。
- 平台对工艺类型标注错误的惩罚：商品下架 + 7 天内整改。

## 4. K 金（不饱和金）

### 4.1 K 金成色

| K 标识 | 含金量 | 千分数 | 常见用途 |
|--------|--------|--------|----------|
| 18K | 75.0% | Au750 | 钻饰主流镶嵌材料 |
| 14K | 58.5% | Au585 | 轻奢 / 设计款 |
| 10K | 41.7% | Au417 | 入门款 |
| 9K | 37.5% | Au375 | 入门款（欧洲常用） |

### 4.2 K 金颜色（合金影响）

| 颜色 | 合金 | 用途 |
|------|------|------|
| **黄 K 金** | Au + Ag + Cu | 传统镶嵌 |
| **白 K 金** | Au + Pd / Ni | 钻石镶嵌（颜色衬托）|
| **玫瑰 K 金** | Au + Cu | 时尚款 |
| **绿 K 金** | Au + Ag | 罕见 |

### 4.3 K 金与"白金"的混淆

> ⚠️ **重要消费者认知误区**：
> - "白金" 在严格意义上指**铂金（Pt）**，而非"白 K 金"。
> - 但消费者口语中常把"白 K 金" 误称为"白金"。
> - 国家标准要求商详页必须明示"白 K 金 / Au750（白色）"，不能简单写"白金"。
> - Agent 在解析规则时遇到"白金" 应主动询问"是 Pt950 铂金还是白 K 金？"。

## 5. 印记 / 印章（电商商详必备）

国家标准要求所有贵金属饰品必须打"印记 / 钢印"，便于识别：

| 项 | 印记 |
|----|------|
| 厂家代号 | 企业X：CTF / 企业X |
| 材料名称 + 纯度 | 足金 / Au999 / 18K / Au750 / Pt950 |
| 镶嵌主石（如有） | DIA（钻石） / RUBY（红宝石） |
| 主石克拉数（如有） | 0.50ct |

> ⚠️ 没有印记 / 钢印的金饰被认定为"三无产品"，平台立即下架且罚款。

## 7. 电商常见黄金活动规则模式

### 7.1 "今日金价立减 X 元/克"

- 含义：当日金价基础上立减 X 元/克。
- 适用：仅计重款。
- Agent 解析：标记为 confidence 0.95（高），输出"克单价 = 基础金价 - X"。

### 7.2 "工费立减 50%"

- 含义：工费按 50% 收取。
- 适用：仅计重款（一口价金通常无工费）。
- Agent 解析：标记 confidence 0.9，输出"工费 = 原工费 × 0.5"。

### 7.3 "古法金 7 折"

- 含义：一口价古法金 SKU 享 7 折活动价。
- 适用：仅古法金一口价 SKU。
- Agent 解析：必须先核验是否 "古法金 = 一口价 = 进入价格管控"。

### 7.4 "以旧换新免手续费"

- 含义：旧金换新款时免工费 / 折损费。
- 一般针对会员或大促节点。

## 8. Agent 应用示例

```text
[活动规则] "企业X古法金传承系列 7 折，全场最低 6 折"
[Agent 解析]
- parsed_rule_1: "古法金（一口价）享 7 折活动价"
- confidence: 0.95
- parsed_rule_2: "全场最低 6 折"
- confidence: 0.6（"全场" 与"古法金 7 折" 范围歧义）
- clarification_question: "全场 6 折是指除古法金外的其他古法金 SKU，还是包含计重款？"

[互斥提示]
- 计重款不应参与"X 折" 类规则，建议运营人员确认。
```
