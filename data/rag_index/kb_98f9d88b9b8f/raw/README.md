# 珠宝行业知识库（Jewelry Industry KB）

> 本知识库提供珠宝行业**通用**领域知识，覆盖黄金、钻石、铂金、银饰、翡翠、珍珠等品类的术语、参数、国家标准与权威鉴定体系。
> Agent 调用本知识库的目的：在 `rule_parser` 节点理解品类术语（如"古法金"、"5G 黄金"、"4C"），在 `checklist_builder` 节点把"克重 / 成色 / 净度" 等参数精度要求转化为可机器核查的检查项。

## 知识库使用原则

1. **作为术语字典使用**：Agent 在 `rule_parser` 时遇到术语，应先在本知识库匹配标准定义。
2. **作为参数精度参考使用**：Agent 在生成检查清单时，应按本知识库定义的参数精度要求（如克重 0.01g、克拉数 0.01ct、净度等级）输出可核查项。
3. **作为风险判定的依据**：Agent 在判定 SKU 是否合规时，应参考本知识库的国家标准条款（如足金 ≥ 99.0%，千足金 ≥ 99.9%）。
4. **作为业务术语翻译使用**：Agent 在向运营人员解释规则时，应优先使用本知识库的"通俗解释"列，避免技术黑话。
