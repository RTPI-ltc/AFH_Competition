# 翡翠、珍珠、和田玉基础知识（Jadeite, Pearl, Jade）

> 适用于 Agent 解析玉石、珍珠类活动规则。

---

## 1. 翡翠（Jadeite）

### 1.1 翡翠 ABC 货

| 等级 | 定义 | 是否合规 |
|------|------|---------|
| **A 货** | 天然翡翠，未经任何人工优化 | ✅ 合规，可销售 |
| **B 货** | 经过酸洗 + 注胶处理 | ❌ 必须明示，不可冒充 A 货 |
| **C 货** | 经过染色处理 | ❌ 必须明示，不可冒充 A 货 |
| **B+C 货** | 酸洗 + 注胶 + 染色 | ❌ 必须明示 |
| **D 货** | 玻璃 / 塑料仿制 | ❌ 必须明示"仿翡翠" |

> ⚠️ 电商必须明示"A 货 / B 货 / 优化处理" 字样；B / C / D 货当作 A 货销售构成欺诈，最高罚款 50 万 + 行政处罚。

### 1.2 翡翠"种 / 水 / 色"

| 维度 | 描述 | 商业价值 |
|------|------|---------|
| **种** | 翡翠晶体的细腻程度 | 玻璃种 > 冰种 > 糯种 > 豆种 |
| **水** | 透光性 | 透 > 通 > 干 |
| **色** | 颜色（绿、紫、黄、红、黑） | 帝王绿 > 苹果绿 > 阳绿 > 油绿 |

### 1.3 翡翠 A 货证书

- **颁发机构**：NGTC / 国检 / 各省珠宝玉石质量监督检验中心。
- **核心字段**：
  - 鉴定结论：翡翠（A 货）
  - 颜色：（具体颜色）
  - 形状 / 重量
  - 折射率 / 密度 / 内部特征
- **平台要求**：天猫 / 京东 / 淘宝必须展示 A 货证书电子件。

## 2. 珍珠（Pearl）

### 2.1 珍珠分类

#### 按生长方式

| 类型 | 定义 | 价格 |
|------|------|------|
| **天然珍珠** | 完全自然生长（极罕见） | 顶级 |
| **养殖珍珠** | 人工植入核 / 养殖 | 中等到高 |

#### 按水域

| 类型 | 主要产地 | 特点 |
|------|---------|------|
| **海水珍珠** | 日本 / 中国 / 大溪地 / 南洋 | 圆度高、光泽强、价高 |
| **淡水珍珠** | 中国浙江 / 安徽 | 形状多变、色彩丰富、价中 |

#### 主流海水珍珠品类

| 品类 | 产地 | 直径 | 颜色 | 价格 |
|------|------|------|------|------|
| **Akoya 珍珠** | 日本 / 中国 | 5–10mm | 白 / 粉 | 中高 |
| **南洋珠（白）** | 澳大利亚 / 印尼 | 9–18mm | 白 / 银 / 金 | 顶级 |
| **南洋珠（金）** | 菲律宾 / 印尼 | 9–16mm | 金 | 顶级 |
| **大溪地黑珍珠** | 法属波利尼西亚 | 8–16mm | 黑 / 绿 / 紫 | 顶级 |

### 2.2 珍珠分级（7 项指标）

| 维度 | 描述 |
|------|------|
| **直径（Size）** | 毫米精度，影响最大 |
| **形状（Shape）** | 圆度（正圆 > 近圆 > 椭圆 > 巴洛克） |
| **颜色（Color）** | 主色 + 伴色（如白色带粉伴色） |
| **光泽（Luster）** | 极强 / 强 / 中 / 弱 |
| **表皮（Surface）** | 无暇 / 微瑕 / 小瑕 / 重瑕 |
| **珠层（Nacre）** | 厚 / 中 / 薄 |
| **匹配（Matching）** | 一串珍珠间的一致性 |

### 2.3 珍珠证书

- **机构**：NGTC / 国检 / 各省珠宝玉石中心。
- **核心字段**：珍珠类型（海水 / 淡水）、直径、颜色、光泽。

### 2.4 珍珠电商常见活动

- "Akoya 珍珠节 8 折"
- "海水珍珠满 3000 减 300"
- 价格管控：一口价款进入"近 X 天最低价"管控。

## 3. 和田玉 / 软玉（Nephrite）

### 3.1 和田玉分类

| 类型 | 颜色 | 商业价值 |
|------|------|---------|
| **白玉 / 羊脂白玉** | 白（顶级羊脂白） | 顶级 |
| **青白玉** | 偏青的白 | 中高 |
| **青玉** | 青绿 | 中 |
| **碧玉** | 翠绿（俄罗斯产较多） | 中 |
| **墨玉** | 黑 | 中 |
| **黄玉** | 黄 | 罕见 / 顶级 |
| **糖玉** | 红糖色 | 中 |

### 3.2 产地差异

| 产地 | 特点 | 商业价值 |
|------|------|---------|
| **新疆和田（山料 / 山流水 / 籽料）** | 籽料最贵 | 顶级 |
| **青海昆仑** | 与和田同矿脉 | 中 |
| **俄罗斯** | 碧玉为主 | 中 |
| **韩国** | 偏黄、偏黑 | 中低 |

### 3.3 和田玉证书

- **机构**：NGTC。
- **重要提示**：国家标准 GB/T 16553-2017 中"软玉" 包含和田玉、青海玉、俄罗斯玉等，证书一般只写"和田玉" 或"软玉"，**不强制写产地**。商详页明示产地的话，需有独立产地溯源证书。

## 4. 玉石珍珠电商常见踩坑

| 踩坑点 | 详细 | Agent 建议 |
|--------|------|------------|
| **B / C 货当 A 货卖** | 翡翠 B 货销售 | 标记 high；下架 + 法务介入 |
| **染色珍珠当天然色卖** | 染色黑珍珠当大溪地黑珍珠 | 标记 high；商详页必须明示 |
| **产地标注无证** | 标注"新疆和田籽料" 但只有"软玉" 证书 | 标记 medium；建议补产地证书或修改商详 |
| **珍珠光泽虚标** | 标"极强" 实际"强" | 标记 medium |
| **重量虚标** | 翡翠克数虚标 | 标记 medium；建议二次称重 |
