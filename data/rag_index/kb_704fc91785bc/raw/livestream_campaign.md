# 直播专场规则知识库（抖音 / 淘宝直播 / 京东直播）

> **聚焦赛题方向 C**：直播专场是与货架电商规则口径差异最大的渠道；商家在选品执行时极易踩坑。

---

## 一、直播专场 vs 货架电商的关键差异

| 维度 | 货架电商（超品日等） | 直播专场 |
|------|-----------------------|----------|
| 价格主体 | 商家 / 平台 | **达人 / 主播** 也可议价 |
| 价格底线 | 近 X 天最低价 | 通常要求"全网最低 + 直播间专享更低" |
| 价格窗口 | 活动期内统一价 | **直播间限时价**（仅直播过程中生效） |
| 库存 | 单 SKU 上链接库存 | 抢购式：限量库存逐步释放 |
| SKU 数量 | ≥ 100 款 | 单场通常 30–80 款（直播节奏决定） |
| 选品 | 商品标签 + 数据筛 | **主播亲选**或商家匹配 |
| 售后 | 商家承担 | 商家承担 + 主播形象担保 |
| 价保 | 15–30 天 | 通常**不享受价保**（直播间价为限时优惠） |

## 二、抖音直播专场可解析规则

### 2.1 商家资质

| 检查项 | 门槛 |
|--------|------|
| 抖音店铺类型 | 旗舰店 / 专营店 |
| 抖音综合体验分 | ≥ 4.6 |
| 商品体验分 | ≥ 4.5 |
| 物流体验分 | ≥ 4.5 |
| 服务体验分 | ≥ 4.5 |
| 近 30 天违规 | 0 次 |

### 2.2 SKU 选品

| 检查项 | 门槛 |
|--------|------|
| 报名 SKU 数 | 单场 30–80（看直播节奏） |
| 单 SKU 近 30 天销量 | ≥ 50 件 |
| 商详页主图视频 | 必备（直播切片用） |
| 抖音独家 SKU | ≥ 20% |

### 2.3 价格管控（抖音独有口径）

| 检查项 | 规则 |
|--------|------|
| 直播间价 | ≤ 抖音全渠道（含商详、商城、其他直播间） |
| 让利幅度 | ≥ 25%（高于其他平台） |
| 直播间限时价 vs 商详价 | 直播间结束后商详价可恢复，但**期间不可降到比直播间更低** |
| 比价口径 | 抖音生态内"近 30 天最低" |
| 价保 | 直播间限时价**通常不享受价保**（须在商详页明示） |

### 2.4 直播间独有规则

| 规则 | 说明 |
|------|------|
| 限时秒杀 | 1–3 分钟内限量库存释放 |
| 满减券 | 主播代发权益券，可叠加店铺券 |
| 退换货 | 7 天无理由（黄金计重款除外） |
| 主播形象担保 | 主播自承担"假一赔十"形象责任 |

## 三、淘宝直播专场可解析规则

| 检查项 | 门槛 |
|--------|------|
| 经营主体 | 旗舰店 / 专卖店 |
| 淘宝直播带货分 | ≥ 80（百分制） |
| 报名 SKU 数 | 单场 30–60 |
| 直播间价 | ≤ 淘宝全渠道最近 30 天最低 |
| 让利幅度 | ≥ 20% |
| 价保 | 限时价不享价保；非限时价享 15 天 |

## 五、Agent 解析直播专场规则的特殊注意

### 5.1 "直播间价"的歧义

```text
[规则] "直播间价不得高于商详价"
[歧义] 商详价是指"近 30 天最低"还是"今日商详挂牌价"？
[Agent 主动澄清] "本规则中'商详价'是指近 X 天最低成交价，还是当日商详挂牌价？"
```

### 5.2 "全网最低"在直播场景的特殊解释

直播间价通常被认为是"限时极限低价"，监管视角下"全网最低"的口径要严格区分：
- 货架电商：与京东、抖音、品牌官网比价
- 直播场景：与所有直播间（含同 SKU 在其他直播间）比价

### 5.3 主播自带预算（坑位费、佣金、平台补贴）的让利归属

```text
[规则] "直播间让利 ≥ 25%"
[Agent 解析]
- 让利构成 = 商家让利 + 主播让利 + 平台补贴
- 若 25% 全部由商家承担 → 利润穿底
- 主动澄清："25% 让利中商家承担多少、主播让利多少、平台补贴多少？"
```

## 六、常见踩坑

| 风险点 | 触发场景 | 风险等级 |
|--------|---------|---------|
| 直播间价低于商详价 X 天最低 | 形成"商详穿底" | high |
| 限时价被消费者要求价保 | 商详页未明示限时价不价保 | high |
| 同款 SKU 多直播间穿仓 | 不同主播给出不同直播价 | high |
| 让利幅度让利归属不清 | 商家与主播让利分摊错算 | medium |
| 抖音独家 SKU 不足 20% | 选品比例不达 | medium |
| 直播间库存 0 后仍挂链接 | 消费者下单失败差评 | medium |
