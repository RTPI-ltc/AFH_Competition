# 铂金与银饰基础知识（Platinum & Silver）

> 适用于 Agent 解析铂金、银饰活动规则。

---

## 1. 铂金（Platinum, Pt）

### 1.1 铂金成色（GB 11887-2012）

| 标识 | 含铂量 | 通俗名 | 印记 |
|------|--------|--------|------|
| Pt950 | 95.0% | 950 铂金 | Pt950 / Plat950 |
| Pt900 | 90.0% | 900 铂金 | Pt900 / Plat900 |
| Pt850 | 85.0% | 850 铂金 | Pt850 |
| Pt990 | 99.0% | 990 铂金 | Pt990 |

### 1.2 铂金 vs "白金"（核心混淆点）

| 项 | 铂金（Pt） | 白 K 金（Au 750 白） |
|----|------------|----------------------|
| 元素 | 铂（Platinum）单质 | 黄金 + 合金 |
| 颜色 | 自然白色 | 镀白 / 抛光后呈白色 |
| 密度 | 21.45 g/cm³（重） | 15.5 g/cm³（轻） |
| 价格（每克） | 比黄金略低，但加工费高 | 与黄金成正比 |
| 用途 | 婚戒、求婚钻戒 | 时尚款、镶嵌 |
| 印记 | Pt950 / Pt900 | Au750 / 18K |

> ⚠️ "白金" 在国标和法律层面专指铂金（Pt），白 K 金不能简称"白金"。

### 1.3 铂金电商定价

- 多数为**计重模式**：成交价 = 克重 × 铂金价 + 工费。
- 铂金价波动相对独立于黄金价，从国际铂金市场（NYMEX / 上海）跟踪。
- 一口价铂金饰品也有，但占比小。

### 1.4 铂金电商常见活动

- "铂金价立减 X 元/克"
- "铂金对戒 8 折"（限一口价款）
- 价格管控：仅适用于一口价款。

## 2. 银饰（Silver, Ag）

### 2.1 银饰成色（GB 11887-2012）

| 标识 | 含银量 | 通俗名 | 印记 |
|------|--------|--------|------|
| S990 | 99.0% | 足银 | S990 / Ag990 |
| S925 | 92.5% | 925 银 / 标准银 / Sterling Silver | S925 / Ag925 / 925 |
| S800 | 80.0% | 800 银 | S800 / Ag800 |

### 2.2 925 银（电商主流）

- **定义**：含 92.5% 银 + 7.5% 其他金属（通常是铜）。
- **特点**：硬度比足银高，可塑性好，适合做精细工艺。
- **价格**：相对便宜（每克数十元），入门级品类。
- **电商定价**：一般为一口价。
- **镀层**：925 银常镀铂、镀金、镀玫瑰金以提升外观。

### 2.3 925 银的常见问题

- **氧化变黑**：925 银易氧化，需定期清洁。
- **过敏**：合金中的铜 / 镍可能导致皮肤过敏。
- **镀层脱落**：镀层使用 6–12 个月后可能脱落，需翻新。

### 2.4 银饰电商常见活动

- "925 银饰品 5 折"（限定品类）
- "925 银 + 镶嵌（碎钻）8 折"
- 价格管控：一口价银饰品同样进入"近 X 天最低价"管控。

## 3. 其他贵金属

### 3.1 钯金（Palladium, Pd）

- **标识**：Pd950 / Pd990
- **用途**：早期作为白金替代品，目前占比小。
- **特点**：比铂金轻，颜色相似。
- **电商**：以"钯金对戒" 等品类小批量销售。

### 3.2 玫瑰金（Au + Cu 合金）

- **不是独立金属**：而是 Au750（18K）+ 铜的合金，呈玫瑰色。
- **印记**：Au750（玫瑰色）/ 18K Rose Gold。
- **电商**：时尚款 / 设计款常用。

## 4. 铂金 / 银饰电商踩坑点

| 踩坑点 | 详细 | Agent 建议 |
|--------|------|------------|
| **混淆"白金 / 白 K 金 / 铂金"** | 商详页用"白金" 表述白 K 金 | 标记 high；要求商详页用 Pt950 或 Au750 |
| **925 银无印记** | 银饰品缺 S925 钢印 | 标记 high；下架整改 |
| **银饰镀层脱落** | 售后投诉集中 | 售后部门关注，提供翻新服务 |
| **铂金价格管控错配** | 计重款误进入"近 30 天最低价" 池 | 标记 medium；建议剔除 |
| **过敏责任** | 925 银导致皮肤过敏 | 商详页明示成分，提供退换货 |
