# 钻石基础知识（Diamond Basics）

> 适用于 Agent 解析镶嵌 / 求婚钻戒等品类活动规则。

---

## 1. 钻石的 4C 分级（国际标准）

### 1.1 Carat（克拉数）

- 1 克拉 = 0.2 克 = 200 毫克。
- 精度要求：精确到 0.01 克拉（即 1 分）。
- 1 克拉 = 100 分（点 / 分）。
- 常见点位：5 分、10 分、20 分、30 分、50 分、70 分、1 克拉、2 克拉、3 克拉、5 克拉。
- **电商商详页要求**：必须精确到 0.01ct，不能用"约 0.5 克拉" 等模糊表述。

### 1.2 Color（颜色）

颜色等级（D 是最白，Z 是最黄 / 棕，依次降低）：

| 等级 | 描述 | 商业价值 |
|------|------|---------|
| D | 极白（无色透明） | 最高级 |
| E | 极白 | 高级 |
| F | 极白 | 高级 |
| G | 优白 | 中高 |
| H | 优白 | 中高 |
| I | 微黄白 | 中 |
| J | 微黄白 | 中 |
| K | 浅黄 | 中 |
| L–Z | 黄至棕 | 低 |

> ✅ 30 分以上钻石商详页必须标注精确颜色等级（D–Z 中的某个字母）。

### 1.3 Clarity（净度）

净度等级（FL 最高，I3 最低）：

| 等级 | 全称 | 描述 |
|------|------|------|
| FL | Flawless | 完美无瑕 |
| IF | Internally Flawless | 内部无瑕 |
| VVS1 | Very Very Slightly Included 1 | 极微瑕 1 |
| VVS2 | Very Very Slightly Included 2 | 极微瑕 2 |
| VS1 | Very Slightly Included 1 | 微瑕 1 |
| VS2 | Very Slightly Included 2 | 微瑕 2 |
| SI1 | Slightly Included 1 | 小瑕 1 |
| SI2 | Slightly Included 2 | 小瑕 2 |
| I1 / I2 / I3 | Included | 瑕疵明显 |

> ✅ 30 分以上钻石商详页必须标注精确净度等级。

### 1.4 Cut（切工）

切工等级（仅适用于圆形明亮式 Round Brilliant Cut）：

| 等级 | 描述 |
|------|------|
| EX (Excellent) | 完美 |
| VG (Very Good) | 非常好 |
| G (Good) | 好 |
| F (Fair) | 一般 |
| P (Poor) | 差 |

**3EX 切工**：切工（Cut）+ 抛光（Polish）+ 对称（Symmetry）三项均为 EX，是高端钻饰常见卖点。

> ✅ 30 分以上圆钻商详页必须标注切工等级；异形钻（公主方、梨形等）GIA 不给 Cut 评级。

### 1.5 4C 价格敏感度（演示用经验值）

- **Carat 影响最大**：克拉数加倍，价格往往涨 2–4 倍。
- **Color 次之**：D 比 H 贵 50–80%（同克数同净度）。
- **Clarity 再次**：VVS1 比 SI1 贵 30–50%。
- **Cut 影响切工溢价**：3EX 比 G 切工贵 20–30%。

## 2. 钻石证书体系（电商必查）

### 2.1 主流证书机构

| 机构 | 全称 | 总部 | 行业地位 | 适用范围 |
|------|------|------|---------|---------|
| **GIA** | Gemological Institute of America | 美国 | 国际权威，最受认可 | 30 分以上裸钻必备 |
| **IGI** | International Gemological Institute | 比利时 | 国际，培育钻常用 | 培育钻 / 部分天然钻 |
| **HRD** | Hoge Raad voor Diamant | 比利时 | 欧洲权威 | 欧洲市场常用 |
| **NGTC** | 国家珠宝玉石质量监督检验中心 | 中国 | 国内权威 | 国内市场必备 |
| **CTF（T MARK）** | 企业X自有标准 | 中国 | 品牌自有切工标准 | T MARK 系列 |

### 2.2 平台对证书的要求

| 平台 | 30 分以上钻石 | 主石碎钻 | 培育钻 |
|------|----------------|---------|--------|
| 天猫 | GIA / NGTC | NGTC（总重证书） | 必须明示"培育钻 / 实验室钻" 字样 |
| 京东自营 | GIA / NGTC | NGTC | 同上 |
| 京东 POP | GIA / NGTC | NGTC | 同上 |
| 淘宝百亿补贴 | GIA / NGTC | NGTC | 同上 |
| 抖音 | GIA / NGTC | NGTC | 同上 |

### 2.3 GIA 证书核心字段（电商必填）

```
Report Number: 2400001234
Shape and Cutting Style: Round Brilliant
Measurements: 4.31 - 4.34 x 2.65 mm
Carat Weight: 0.30
Color Grade: F
Clarity Grade: VVS2
Cut Grade: Excellent
Polish: Excellent
Symmetry: Excellent
Fluorescence: None
```

### 2.4 培育钻（实验室钻 / Lab-grown Diamond）

- **定义**：通过 HPHT / CVD 工艺在实验室制造的钻石，分子结构与天然钻完全相同。
- **国家标准**：GB/T 16554-2017《钻石分级》同样适用，但商详页必须**明确标注"培育钻 / 实验室钻"**。
- **价格**：约为同 4C 天然钻的 20–30%。
- **争议**：电商中混淆"培育钻" 与"天然钻" 是高频违规行为，监管严打。
- **证书**：IGI 培育钻证书最常见。

## 3. 镶嵌工艺

### 3.1 主流镶嵌方式

| 镶嵌方式 | 中文名 | 特点 |
|----------|--------|------|
| **Prong / Claw Setting** | 爪镶 | 用 4 ~ 6 个金属爪固定，常见六爪经典款 |
| **Bezel Setting** | 包镶 | 金属环绕主石，更安全，但显小 |
| **Pavé Setting** | 密镶 | 小钻密集排列，常见"钻石路面" |
| **Channel Setting** | 槽镶 | 主石嵌入金属沟槽 |
| **Tension Setting** | 张力镶 | 两侧金属夹住主石，无爪 |
| **Halo Setting** | 群镶 / 光环 | 主石外围一圈碎钻 |

### 3.2 镶嵌金属选择

- **Pt950 铂金**：强度高，对钻石颜色衬托好。
- **18K 白金（Au750 白）**：高度抛光后接近铂金视觉效果。
- **18K 玫瑰金（Au750 玫瑰）**：时尚款常用。
- **18K 黄金（Au750 黄）**：复古款常用。

## 4. 钻石电商常见活动规则

### 4.1 "30 分以上钻戒 8 折"

- Agent 解析：限定 carat ≥ 0.30。
- 数据校验：从 SKU 数据中筛选 gem_carat ≥ 0.30。

### 4.2 "GIA 钻戒 7.5 折"

- Agent 解析：限定 certificate.type = "GIA"。
- 风险：需注意是否包含 IGI / NGTC 证书钻戒。

### 4.3 "T MARK 品牌切工系列享额外 95 折"

- Agent 解析：限定 brand = "T MARK"。
- 与全场 8 折叠加？需澄清。

### 4.4 "求婚钻戒以旧换新差额计算"

- 含义：旧戒可抵扣新戒价款；钻石按"鉴定时间 5 折回收 + 同款升级" 公式。
- Agent 解析：confidence 较低（业务规则复杂），建议人工确认。

## 5. 钻石电商常见踩坑点

| 踩坑点 | 详细 | Agent 建议 |
|--------|------|------------|
| **培育钻虚标天然钻** | 商详页未明示"培育钻" 字样 | 标记 high；建议法务核查 |
| **碎钻总重虚标** | 商详页标注"含 0.5ct 钻石" 实际是碎钻总重 | 标记 medium；建议商详页明示"主石 + 副石 + 总重" |
| **30 分以下缺证书** | 30 分以下不强制 GIA，但平台仍要求 NGTC 总重证书 | 标记 medium |
| **切工等级误标** | 异形钻 GIA 不给 Cut，但商详页虚标"3EX" | 标记 high；建议下架整改 |
| **荧光等级隐藏** | 强荧光（Strong）会影响钻石透光，需在商详页标注 | 标记 medium |
