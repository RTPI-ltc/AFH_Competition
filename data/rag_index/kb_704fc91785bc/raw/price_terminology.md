# 价格术语统一字典（天猫 / 淘宝 / 京东 / 抖音通用）

> 本文档统一定义价格相关术语，供 Agent 在解析活动规则时做"语义归一化"。
> Agent 解析规则时如果遇到术语，应优先匹配本文档中的标准定义；若规则原文与本文档定义冲突，以**规则原文为准**，并在 `risk_points` 中标记"术语口径冲突"。

---

## 1. 价格术语速查表

| 术语 | 标准定义 | 平台 / 场景差异 | 合规要点 |
|------|---------|-----------------|----------|
| **吊牌价** | 商品物理吊牌上的标价，通常等于品牌指导零售价 | 各平台通用 | 不能直接作为商详页"划线价"使用 |
| **品牌指导价 / 厂商指导价（MSRP）** | 品牌方建议的统一对外零售价 | 各平台通用 | **禁止**用作划线价（虚标参考价行为，可处罚） |
| **店铺销售价 / 日常价** | 店铺对外正常售卖的价格（非活动期） | 各平台通用 | 必须是近期有实际成交、未被划线的价格 |
| **划线价 / 原价** | 商详页中带删除线展示的"原价" | 天猫近 30 天 / 京东近 14 天 | 必须等于近期有成交、未划线的销售价 |
| **活动价 / 促销价** | 活动期间实际成交价（不含跨店满减、红包） | 各平台 | 一般 ≤ 近 30 天最低成交价 |
| **到手价** | 用户实付价格 = 活动价 - 跨店满减 - 红包 - 优惠券 | 各平台 | 详情页"到手 xxx 元"必须按"最大优惠组合"计算 |
| **参考价** | 一般用于商详非主价位置，标注"参考某渠道价" | 通用 | 必须真实存在该渠道，不能虚标 |
| **预售价 / 定金膨胀价** | 定金 + 尾款的总价，膨胀部分由定金倍数决定 | 双 11、6.18 多用 | 定金一般 50–500 元，膨胀 2–5 倍 |
| **直降价** | 不通过任何券和满减，直接降到的价格 | 通用 | 必须是与日常价相比的真实差额 |
| **专享价 / VIP 价 / PLUS 价** | 仅限会员可见 / 可享的价格 | 88VIP / PLUS / 京东 PLUS | 不要求"全网最低"，但要求"会员中最低" |
| **京采价** | 京东自营对品牌方的供货价（内部口径） | 仅京东自营 | 不对外展示 |
| **百亿补贴价** | 进入百亿补贴池的对外最终价 | 淘宝/京东/拼多多各家定义 | 必须 ≤ 全网最低 |
| **官方立减 / 平台立减** | 平台直接对消费者补贴的让利 | 各平台百亿补贴 | 商家不出资部分由平台承担 |
| **历史最低价 / 近 30 天最低价 / 近 90 天最低价 / 近 365 天最低价** | 指定回溯窗口内的店内最低成交价 | 各平台口径不同 | **天猫 30 天 / 京东 14 天 / 百补 90 天** |
| **保留价** | 商家给平台的最低可接受成交价 | 京东自营 / 聚划算入驻谈判 | 不对外公开 |

## 2. "近 X 天最低价"口径对照表（最容易踩坑）

不同平台对"最低价"的回溯窗口与计算方式不同，Agent 解析时必须按平台区分：

| 平台 / 频道 | 回溯窗口 | 计算口径 | 备注 |
|-------------|----------|---------|------|
| 天猫超级品牌日 | 近 **30 天** | 店铺 SKU 实际成交价（不含跨店满减、红包） | 取最低单日成交价 |
| 天猫 11.11 / 6.18 | 近 **30 天** + 近 **365 天** 双重校验 | 同上 | 部分大促对历史最低价再降 5% |
| 京东超级品牌日 | 近 **14 天** | 店铺 SKU 实际成交价 | 京东独有 14 天口径 |
| 京东百亿补贴 | 近 **90 天** | 店铺 SKU 实际成交价 + 全网比价 | 必须同时是全网最低 |
| 淘宝聚划算品牌闪购 | 近 **30 天** | 实际成交价 | 与天猫一致 |
| 淘宝百亿补贴 | 近 **90 天** + **全网比价** | 同上 | 全网包含京东、抖音、拼多多、得物、品牌官网 |
| 抖音商城 | 近 **30 天** + **抖音生态内比价** | 抖音生态内最低 | 抖音"低价心智频道" 要求生态内全网最低 |
| 拼多多 | 近 **30 天** | 拼多多平台内最低 | 不与外渠对比 |
| 得物 | 近 **30 天** | 得物平台内最低 | 鉴别后实付价 |

> ⚠️ "近 X 天" 是滚动窗口还是自然月？业内标准是**滚动窗口**（活动开始日往前推 X 天），但部分商家会按"自然月"理解，导致价格穿底。Agent 在解析规则时若发现"近 30 天"表述含混，应主动生成澄清问题。

## 3. 划线价合规专题

### 3.1 国家市场监督管理总局《明码标价和禁止价格欺诈规定》（2022 年 7 月起施行）核心要点

1. 划线价（被划线的"原价"）必须是"促销前 7 日内最低成交价"。
2. 若 7 日内无成交记录，应按"最后一次成交价 / 最近一次销售价" 标注。
3. 不得使用"建议零售价 / 厂商指导价 / 吊牌价" 作为划线价（除非有充分依据）。
4. 不得使用虚高原价误导消费者；处罚最高 50 万元 + 行政警告。

### 3.2 各平台对划线价的额外要求

| 平台 | 要求 |
|------|------|
| 天猫 | 近 30 天有实际成交且未划线的销售价 |
| 京东 | 近 14 天有实际成交且未划线的销售价 |
| 淘宝 C 店 | 近 7 日有实际成交（参照市监总局规定） |
| 抖音商城 | 近 30 天 + 行业大促期间再加 7 天保护期 |

### 3.3 常见违规划线价

- ❌ 使用 5 年前的吊牌价作为划线价。
- ❌ 把"建议零售价 9999 元" 作为划线价，实际从未以该价销售。
- ❌ "活动价 199 元，原价 999 元"，但近 30 天该 SKU 实际最高成交价仅为 399 元。
- ✅ 划线价 399 元（近 30 天最高有成交）、活动价 199 元，合规。

## 4. 让利幅度与补贴口径

### 4.1 商家让利幅度计算

```
让利幅度 = (划线价 - 活动价) / 划线价
让利金额 = 划线价 - 活动价
```

### 4.2 到手价让利幅度计算

```
到手让利幅度 = (划线价 - 到手价) / 划线价
到手价 = 活动价 - 跨店满减 - 平台红包 - 店铺券 - 平台立减
```

### 4.3 平台对让利幅度的门槛（参考值）

| 频道 | 让利门槛 |
|------|---------|
| 天猫超品日 | ≥ 15%（含让利、券、平台补贴合计） |
| 京东超品日 | ≥ 15% |
| 淘宝百亿补贴 | ≥ 15% |
| 京东百亿补贴 | ≥ 20% |
| 抖音超值购 | ≥ 25%（抖音对让利门槛要求更高） |

## 5. 价格相关常见歧义（供 Agent 主动澄清）

| 歧义点 | 可能解读 1 | 可能解读 2 | Agent 建议 |
|--------|------------|------------|------------|
| "近 30 天最低价" | 自然月 | 滚动 30 天 | 主动澄清，默认按滚动 30 天 |
| "全网最低价" | 全平台 | 仅天猫 / 仅京东 | 主动澄清，默认按全平台 |
| "到手价" | 含跨店满减 | 不含跨店满减 | 主动澄清，默认含跨店满减 |
| "活动价 ≤ 历史最低" | 近 365 天 | 历史所有时段 | 主动澄清，默认近 365 天 |
| "店铺券可叠加" | 与跨店满减叠加 | 与百亿补贴叠加 | 主动澄清，默认与跨店满减叠加 |
| "划线价" | 原价 / 吊牌价 | 近 30 天最高成交价 | 主动澄清，按市监总局规定 |
