# 平台审核驳回原因案例库（Audit Rejection Cases）

> Agent 在 `rule_parser` / `checklist_builder` 节点除了解析正向规则，还应预判"平台可能因为什么原因驳回报名"。本文档整理常见驳回原因 + 整改路径，作为 `risk_points` 的补充输出来源。

---

## 1. 资质类驳回

### 1.1 营业执照 / 商标证临期或过期

| 项 | 内容 |
|----|------|
| 触发场景 | 报名时证件距到期 < 30 天 |
| 平台反馈典型话术 | "贵店营业执照临近到期，请先完成年审" |
| Agent 风险等级 | high（已过期）/ medium（临期） |
| 整改路径 | 法务先完成年审 → 上传新证 → 重新报名 |
| 整改时效 | 3–7 工作日 |

### 1.2 贵金属生产 / 销售许可缺失

| 项 | 内容 |
|----|------|
| 触发场景 | 报名珠宝类目活动，但未上传贵金属相关许可 |
| 平台反馈 | "请补充贵金属行业相关资质证明" |
| Agent 风险等级 | high |
| 整改路径 | 商务对接 → 法务出具 |
| 整改时效 | 1–3 工作日 |

### 1.3 ISO / RJC 等推荐性认证缺失

| 项 | 内容 |
|----|------|
| 触发场景 | 部分平台（如京东自营、抖音品牌严选）要求 ISO 9001 |
| 平台反馈 | "建议补充 ISO 9001 等行业体系认证" |
| Agent 风险等级 | medium |
| 整改路径 | 视活动重要性决定是否补办 |

---

## 2. 价格类驳回（最高频）

### 2.1 活动价高于近 X 天最低价

| 项 | 内容 |
|----|------|
| 触发场景 | 系统自动比对订单库 |
| 平台反馈 | "SKU XX 活动价高于近 30 天最低成交价 188 元，请调整" |
| Agent 风险等级 | high |
| 整改路径 | 价格管理调价 → 商务 / 品类经理审批让利 → 重新提报 |
| 整改时效 | 24 小时内 |

### 2.2 划线价虚高

| 项 | 内容 |
|----|------|
| 触发场景 | 划线价不是"近 X 天有实际成交的非划线销售价" |
| 平台反馈 | "划线价不符合《明码标价和禁止价格欺诈规定》" |
| Agent 风险等级 | high |
| 整改路径 | 改为近 X 天最高有成交的销售价 |

### 2.3 让利幅度不足门槛

| 项 | 内容 |
|----|------|
| 触发场景 | 让利 < 15%（超品日）/ < 25%（抖音直播）/ < 10%（品类日） |
| 平台反馈 | "让利幅度未达本期次门槛 15%，建议增加店铺券或调低活动价" |
| Agent 风险等级 | high |
| 整改路径 | 加店铺券 / 降活动价 / 与平台谈补贴 |

### 2.4 全网比价穿底

| 项 | 内容 |
|----|------|
| 触发场景 | 风控算法抓取到其他平台同款更低价 |
| 平台反馈 | "贵店 SKU XX 在京东 / 抖音 / 拼多多发现更低价 198 元，请拉平" |
| Agent 风险等级 | high |
| 整改路径 | 跨平台同步降价 / 或退出本期百补 |

---

## 3. 商品 / SKU 类驳回

### 3.1 销量 / 好评率不足

| 项 | 内容 |
|----|------|
| 触发场景 | 近 90 天销量 < 50 件、好评率 < 95% |
| 平台反馈 | "贵店 SKU XX 销量不达本期次门槛" |
| Agent 风险等级 | high |
| 整改路径 | 替换为达标 SKU |

### 3.2 商品参数不全

| 项 | 内容 |
|----|------|
| 触发场景 | 30 分以上钻石未填完整 4C；黄金成色未填 |
| 平台反馈 | "请完善商品基本信息" |
| Agent 风险等级 | medium |
| 整改路径 | 补全后重报 |

### 3.3 商品证书电子件缺失

| 项 | 内容 |
|----|------|
| 触发场景 | 30 分以上钻戒缺 GIA / NGTC 证书；翡翠缺 A 货证书 |
| 平台反馈 | "请上传可在线核验的商品证书电子件" |
| Agent 风险等级 | high |
| 整改路径 | 从 CMS 拉取或补发 |

### 3.4 培育钻当天然钻

| 项 | 内容 |
|----|------|
| 触发场景 | 商详页未明示"培育钻 / 实验室钻" |
| 平台反馈 | "培育钻商品须在商详页明显位置明示" |
| Agent 风险等级 | high |
| 整改路径 | 商详页整改 + 法务介入 |

### 3.5 翡翠 B/C 货当 A 货

| 项 | 内容 |
|----|------|
| 触发场景 | 商详页未明示"A 货" / 实际为 B 货销售 |
| 平台反馈 | "存在违反国标 GB/T 16552 风险" |
| Agent 风险等级 | high（属于欺诈） |
| 整改路径 | 立即下架 + 法务自查 |

## 4. 库存类驳回

### 4.1 库存承诺与实际差距过大

| 项 | 内容 |
|----|------|
| 触发场景 | 承诺 500、实际 100 |
| 平台反馈 | "请提供真实库存承诺，差距过大将影响活动效果" |
| Agent 风险等级 | high |
| 整改路径 | 调拨补货 / 下调承诺 |

### 4.2 库存承诺不达门槛

| 项 | 内容 |
|----|------|
| 触发场景 | 承诺 200 < 门槛 500 |
| 平台反馈 | "未达本期次最低库存承诺" |
| Agent 风险等级 | high |
| 整改路径 | 调拨补货或下架该 SKU |

---

## 5. 互斥类驳回

### 5.1 已参加互斥活动

| 项 | 内容 |
|----|------|
| 触发场景 | 系统检测同 SKU 已在互斥活动报名 |
| 平台反馈 | "SKU XX 已参加品牌日 / 百亿补贴，本期次不可重复报名" |
| Agent 风险等级 | high |
| 整改路径 | 二选一 |

### 5.2 跨子品牌店铺重复

| 项 | 内容 |
|----|------|
| 触发场景 | 同集团 LOLA、SOINLOVE、T MARK 等同步报名 |
| 平台反馈 | "同集团多店铺报名需平台许可" |
| Agent 风险等级 | medium |
| 整改路径 | 集中到主店铺或商务申请豁免 |

---

## 6. 商详 / 配置类驳回

### 6.1 商详页价保规则未明示

| 项 | 内容 |
|----|------|
| 触发场景 | 价保规则未在商详页醒目位置标注 |
| 平台反馈 | "请在商详页明示价保规则" |
| Agent 风险等级 | medium |
| 整改路径 | 商详配置整改 |

### 6.2 商详页 7 天无理由错配

| 项 | 内容 |
|----|------|
| 触发场景 | 黄金计重款标"7 天无理由"，实际不支持 |
| 平台反馈 | "黄金计重款若不支持 7 天无理由请明示" |
| Agent 风险等级 | high |
| 整改路径 | 商详整改 |

### 6.3 满赠估值超 30%

| 项 | 内容 |
|----|------|
| 触发场景 | 满赠中赠品价值 > 商品本身 30% |
| 平台反馈 | "满赠估值过高视为变相打折" |
| Agent 风险等级 | medium |
| 整改路径 | 拆为"满赠 + 直降"双结构 |

## 7. Agent 整改路径输出格式

每条 `risk_point` 应包含整改路径（基于本案例库）：

```json
{
  "rejection_pattern": "活动价高于近 30 天最低价",
  "platform_feedback_sample": "SKU CTF-FJ-999-002 活动价 2680 高于近 30 天最低 2580，请调整",
  "severity": "high",
  "remediation": {
    "owner": "价格管理 + 品类经理",
    "actions": [
      "调价至 ≥ 2580 元",
      "或商务向平台申请豁免",
      "或退出本期报名"
    ],
    "sla_hours": 24
  }
}
```

## 8. 案例库使用说明

- 黑客松版本：本案例库静态写死，覆盖典型场景。
- 产品化版本：接入平台真实驳回邮件 / 后台消息，做 NLP 归类，自动扩充本案例库。
- 历史驳回记录可作为 `auto_decision_planner` 节点"主动预判风险"的训练数据。
