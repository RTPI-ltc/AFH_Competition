# 商品资质（Product Certifications）

> 本文档定义企业X各品类商品在电商平台报名活动时**必须具备的证书与电子件**。Agent 在选品时按本表逐 SKU 核对，缺失任意一项即标记为不合规。

## 2. 平台对证书的硬性要求

### 2.1 天猫

- **黄金类**：商详页必须展示"CMA / CNAS 实验室抽检报告" 至少 1 份，且报告生成日期距活动日 ≤ 6 个月。
- **钻石类**：30 分及以上必须有 GIA / NGTC 证书；商详页必须展示腰码或证书图片。
- **翡翠类**：必须展示 NGTC A 货证书。
- **超品日 / 百补**：要求"每个报名 SKU 必须有可在线核验的证书电子件"。

### 2.2 京东

- **自营**：京东买手对证书审核更严，要求每个 SKU 至少 1 份国检证书电子件。
- **POP**：与天猫一致，要求 CMA / NGTC 证书。
- **京东品质**：贴有"京东品质" 标签的商品需通过京东自有质检（额外抽检）。

### 2.3 淘宝百亿补贴

- 与天猫一致，要求 GIA / NGTC / CMA。
- 全网比价时会附带证书核验。

### 2.4 抖音

- 黄金计重款必须展示"克重 + 工费 + 实时金价" 三者拆分。
- 钻石类要求 30 分以上 GIA 证书 + 抖音商品体验分 ≥ 4.5。

## 3. 企业X内部证书管理流程

### 3.1 证书入库流程

```
1. 工厂出厂 → 内部 QC 检测 → 出厂合格证
2. 抽检送检 → CMA / NGTC 实验室 → 证书 PDF + 编号
3. 证书入"证书管理系统"（CMS）→ 与 SKU 绑定
4. 电商上架时从 CMS 拉取证书电子件 → 商详页展示
5. 证书有效期满 → 自动触发抽检任务
```

### 3.2 证书缺失的处理

| 场景 | 处理动作 |
|------|---------|
| 全新 SKU 未送检 | 上架前必须完成抽检；阻塞活动报名 |
| 证书有效期临期（< 30 天） | 立即送检，避免活动期间过期 |
| 证书有效期已过 | 下架 SKU，不参与活动 |
| 证书 PDF 文件丢失 | 联系实验室补发，48 小时内出件 |
| 跨平台证书要求不一致 | 以最严平台为准（一般是抖音 / 京东自营） |

## 5. 常见证书风险点（Agent 输出参考）

| 风险点 | 触发条件 | 风险等级 |
|--------|---------|---------|
| 证书有效期 < 30 天 | 临期 | medium |
| 证书有效期已过 | 已过期 | high |
| 30 分以上钻石缺 GIA 证书 | 缺证 | high |
| 翡翠类缺 A 货证书 | 缺证 | high |
| 商详页未展示证书图片 | 配置缺失 | medium |
| 国检批次报告 > 6 个月 | 抽检周期超期 | medium |
| 全网证书编号不一致 | 多平台数据错配 | high |
